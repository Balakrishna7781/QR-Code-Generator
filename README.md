# QR Code Generator

## Overview
A Python application that generates a QR code image from a given URL.  
Built for MSCS-633-A01 Advanced Artificial Intelligence — Hands-On Assignment 2.

**QR CODE LINK:** https://www.bioxsystems.com

## Prerequisites
- Python 3.10+
- pip install qrcode[pil]

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python QR_generator.py
```

## Output
QR Code successfully generated: qr_code.png
QR CODE LINK: https://www.bioxsystems.com

## Files
| File | Description |
|---|---|
| `QR_generator.py` | Main Python script |
| `requirements.txt` | Dependencies |
| `qr_code.png` | Generated QR code output |
